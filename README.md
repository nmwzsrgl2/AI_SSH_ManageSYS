# 智能运维管理系统

一个基于本地 AI 的安全运维管理平台，集服务器监控、网络设备巡检和智能问题分析于一体。

## 功能

- **远程控制**：WebSocket + SSH 加密连接主机，MCP 限制命令执行，仅查看日志、配置和系统状态。
- **智能助手**：
  - 本地微调 AI 模型 + 知识库
  - AI 对话框实时交互，提供巡检分析和问题解答
  - Dify AI Agent 根据对话内容自动选择 LLM 或 Agent
- **前端仪表盘**：
  - 本机性能监控（CPU、内存、硬盘）
  - 服务器日志查看
  - Zabbix 主机状态通过 API 获取
  - 用户管理（增删改查）
- **网络设备巡检**：
  - SSH 连接华为设备执行脚本
  - 保存执行结果到数据库，并返回前端展示
  - AI 可自动分析巡检结果
- **数据库**：SQLite + SQLAlchemy 管理用户和巡检数据

## 技术架构

| 模块     | 技术/工具                  |
| -------- | -------------------------- |
| 前端     | Web + AI 对话框            |
| 后端     | Flask + Flask-Session      |
| AI       | 本地微调大模型，Dify Agent |
| SSH 管理 | FastMCP，受限命令          |
| 监控     | Zabbix API                 |
| 数据库   | SQLite + SQLAlchemy        |

## 安全设计

- 受限 SSH 命令，避免危险操作
- Flask Session 认证，保证用户安全
- 后端严格控制数据访问

## 部署步骤

1. 配置本地微调 AI 模型并接入知识库
2. 安装 Flask 和依赖，初始化 SQLite 数据库
3. 部署 MCP 并设置命令权限
4. 部署前端仪表盘，连接后端和 Zabbix API
5. 启动 Flask 后端服务，确保 Session 认证生效
6. 启动 WebSocket 服务，实现远程 SSH 控制
